Mary's Magical Adventure is a first person platforming game where a seemingly ordinary little girl named Mary must escape Hell City, her home, after seeing a giant human-like silhouette who calls himself the Origin Master appear and throw a fireball too close to her home!

See the World of Kirbycraft Wiki for additional details.

  • Run through areas, killing monsters, finding special coins with unique temporary bonuses, and discover secret areas.
  • Find Special Stage entrances and collect 7 Chaos Bridges.
  • Eight worlds of four levels each planned!

    • Additionally, to listen to the soundtrack, visit Xane's SoundCloud.

    Downloads

    You appear to be on a mobile device. Ignore the downloads below and follow these steps if you haven't done them before but would rather play on a phone or tablet over a computer.

  • Contact Beloko Games. Ask him if you can become a beta tester for Delta Touch. If you're accepted into the Google+ community, purchase Delta Touch on Google Play for ~$3.
  • Install Delta Touch and switch to GZDoom 3.2 by tapping the arrow icon in the upper-right corner.
  • Choose Freedoom: Phase 2 from the menu then tap the play button. When offered to download soundfonts and the Freedoom IWAD, accept. Connect to Wi-Fi if on a limited mobile data connection.
  • Download the mobile game data and extract the files to OpenTouch/Delta/mods.
  • Return to Delta Touch and tap the button in te lower right corner, which displays your mods. Select all three files which start with "mma".
  • Make sure Freedoom: Phase 2 is selected then tap the play button. If a video displaying a spinning blue cube or the letter "A" appears in the upper left corner, you have successfully downlaoded this game.

    If you're on a computer and are seeing this message by mistake, zoom out this page or increase your resolution. If neither work, just download the executable and computer game data below for your OS.

  • If you are a beta tester of Delta Touch on Android (or iOS whenever that version's released), download the mobile game data below and extract the files to OpenTouch > Delta > mods. After that, load them as mods in Delta Touch with Doom, Doom 2, or Freedoom: Phase 2 as the IWAD.
  • If you're on a variant of Linux that isn't based on Ubuntu or a derivative or just want to compile the source yourself, click the box below, find the source link, then clone/fork the repository or download it as a ZIP, then compile using the instructions found on ZDoom's wiki. Don't forget to move the game data to the compiled app's folder once it compiles!
    • Info about security warnings

    Mary's Magical Adventure is not well-known and doesn't have a security certificate to sign officially-released executable files with. Also, anti-viruses are quick to declare unpopular apps as viruses without second thought, so this sectuion will tell you the possible problems you may encounter when initially getting Mary's Magical Adventure to run.

  • If you're using Norton, it may detect this game as a virus named WS.Reputation1. If it does, tell it to restore the play.exe file. Viruses would never be inserted into this game!
  • Windows SmartScreen will warn you about the app/program not having a publisher when you first run it. To avoid this, right-click play.exe and in properties check the checkbox that disables blocking. If greeted with SmartScreen, choose "more info" then "Run Anyways".
  • If MacOS's Gatekeeper prevents the game from running, temporarily disable it in System Preferences.

    Mods

    Mary's Magical Adventure accepts optional downloads developed by Xane which add or change things in-game! Click/tap the screenshot beside the mod you want to download.

    If a mod doesn't work correctly, try:

  • Download the precompiled build or download/compile the source code again.
  • Download the game data.
  • Redownload all mods to make sure they are up to date with the base game data.
  • Make sure the executable, game data, and mods are all in the same folder.

    • If none of this makes the mods work, contact Xane.
