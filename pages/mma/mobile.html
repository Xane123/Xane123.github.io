Mary's Magical Adventure is a first person platforming game where you can experience a father and his daughter's attempts to eliminate two evil men that have tried to get rid of him in the past.

See the Xane Wiki for more details.

  • Are you skilled enough? Quickly speed through areas, clearing them of enemies and finding hidden areas! Earn the rewards.
  • Find entrances to the mysterious six Special Stages, which are said to store the powerful Chaos Bridges, should you be quick about it!
  • Four worlds with four levels in each are planned, each with their own boss!

Additionally, to listen to the soundtrack, visit Xane's SoundCloud.

Downloads

Although Mary's Magical Adventure is designed to be played on a computer, you can play it on Android. It requires more steps and a single purchase, however; Follow these instructions to play MMA on the go:

How to Setup Mary's Magical Adventure on Android
  • Visit Google Play and search for Delta Touch by Open Touch Games, which costs $3.49. After purchasing it, install it on your mobile device.
  • Open Delta Touch and use the arrow in the upper-right corner of the screen to switch to GZDoom v3.3pre.
  • Download the game data from the link below along with the mobile mod, which tweaks things for touch controls and lower-end mobile devices.
  • Once those are downloaded, use a file manager app to extract the files from both ZIP files. (In FX, you have to select each file in each ZIP file then choose "Download" from the upper-right menu.)
  • If prompted for a location to extract to, choose SD card/OpenTouch/Delta/mods. If the last folder doesn't exist, create it. (In FX, select all of the files that were "downloaded" to the Downloads folder, choose "cut" at the top of the screen", then navigate to the aforementioned "mods" folder.)
  • Return to Delta Touch. Tap "Freedoom: Phase 2" from the IWAD list, then tap the keyboard button at the bottom right corner of the screen.
  • A dialog box will appear. Tap the three-line button then select the game data files in any order, as long as mma_data.pk3 comes before the others.
  • Exit the dialog box then tap the play button in the lower-right corner. You'll have to pass the Google Play license check, download Freedoom: Phase 2, and download 70 megabytes of soundfonts, the last one not mattering for now unless a MIDI soundtrack is ever added. Allow all of those, tapping the play button after each is finished. Connect to Wi-Fi if on a limited data plan.
  • After those files all download, tap the play button again. You should see Mary drinking from a restaurant cup then the Xane Corp. logo should appear if everything was done correctly. If Freedoom: Phase 2's title screen appears instead of the Xane Corp. logo, stop the game then re-select the game data/mods again using the three-line button as before.

After this, it's up to you to configure the touch controls and ZDoom settings. Some settings need to be changed to the different defaults the computer version uses for consistency, but that'll be written about later on.

Download
Mods

Mary's Magical Adventure accepts optional downloads developed by Xane which add or change things in-game! Click/tap the picture beside the mod you want to download.

If a mod doesn't work correctly, re-download the game data and the mods. Make sure the mod is selected in the mods list in Delta Touch, too!

Mobile Modifications

Changes some text and sounds for more accurate touchscreen control instructions. Go to Options (F4) > Miscellaneous Options > Graphical Priority and change it to performance for lower-end devices.

Materials

Adds shiny metal and wood surfaces that reflect nearby lights to the game. Only use this if you have a recent graphics card that can handle them. Delta Touch/Android doesn't display these correctly, so only use this mod on a computer.

Hi-res Textures

Increases the resolution of textures and weapon sprites. Recommended in most scenarios.
