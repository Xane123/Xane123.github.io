Text Codes

Code Action
<b> Make Text BOLD.
<i> Makes text ITALIC.
<u> UNDERLINES text.
<div align="left/center/right"> Aligns text to a side of the screen. The default side is left. Change the side by inserting this code.
<font size="(number)"> Changes the font size.
</a> Ends Link, even the <a name=(NAME)> code.
<BLOCKQUOTE> AND </BLOCKQUOTE> Unknown Meaning. Used on CheatCC Forums as a code for exact quotes.
<h1>, <h2>, and so on... MAKES TEXT A HEADING!!!
<bgsound id="Name (name.wav)"> and
<script language="JScript" and something else>		 Plays a sound. Cannot be used if ⁢div id="HelpMsg" class="hide"> is there.
